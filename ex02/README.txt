В проекте используется база данных - PostgreSQL. Создается пустая таблица. Создаются студенты класса Student.
Добавляются в таблицу с помощью sql запроса INSERT. Выводятся из таблицы в stdout через SELECT. Удаляется из таблицы 1 id и снова выводится на экран.
В resourses лежат некоторые файлы для ознакомления и тестирования. В проекте они не используются.


Должен быть установлен JDK. Необходимо открыть проект как Maven, запустить сервер базы данных postgreSQL локально,
указать порт 5434, user: postgres, password: postgres.
Далее запустить код через IDE (я использую IntelliJ Idea).
